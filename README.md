# 🚀 Custom Animated GitHub Stats API

A high-performance, edge-runtime API built with **Next.js** and **TypeScript** to generate gorgeous, animated, and glassmorphism-style metrics cards for your GitHub profile README.

## ✨ Features

- **Frosted Glass UI:** Modern glassmorphism design that blends perfectly with GitHub's dark mode.
- **Neon Cyan Theme:** Consistent professional color scheme based on premium UI trends.
- **Live SVG Animations:** Smooth CSS-based animations for progress bars, pulsing flames, and text fade-ins.
- **Edge Runtime:** Powered by Vercel Edge Functions for ultra-fast, global response times.
- **Real-time Data:** Fetches live data directly from GitHub's GraphQL API.

---

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **API:** GitHub GraphQL API v4
- **Deployment:** [Vercel](https://github-profile-metrics-two.vercel.app/)
